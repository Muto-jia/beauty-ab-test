# 美妆电商用户行为 A/B 测试项目
基于阿里云天池公开美妆用户行为数据集，完成推荐策略AB全流程分析，**MySQL+Python**全链路数据分析项目。

## 📌 项目背景
电商首页商品推荐排序直接影响用户转化，本次实验对比两套商品排序策略的业务效果：
- **对照组A**：按用户浏览点击率排序
- **实验组B**：按用户加购、购买意向权重排序
通过用户哈希分流、指标计算、统计显著性检验，验证新策略是否业务有效。

## 📊 数据集说明
- 数据集来源：**阿里云天池公开数据集 - 美妆用户行为数据集**
- 数据字段：用户ID、商品ID、用户行为类型(1浏览/2收藏/3加购/4购买)、商品品类、日期、小时、用户地域
- 数据范围：2023年双十一、双十二大促脱敏用户行为数据
- 🔔 注意：**原始CSV数据集不上传仓库**，可自行前往阿里云天池检索下载，本地导入MySQL即可复现项目

## 🧪 实验设计
1.  **分流方式**：基于`user_id`哈希随机5:5分流，保证同一用户仅进入单一实验组，无交叉污染
2.  **核心指标**
    - 购买转化率
    - 加购率
    - 人均GMV（人均购买次数近似）
3.  **辅助指标**
    - 收藏率
    - 人均浏览商品数
4.  **统计检验方法**
    - 比例类指标（转化率）：卡方检验
    - 均值类指标（人均GMV）：独立样本t检验
    - 显著性水平：$\boldsymbol{\alpha=0.05}$

## 📈 实验结果与结论
1.  指标对比：A、B两组所有业务指标数值差异极小
2.  显著性检验结果：**所有指标p值均>0.05，差异不显著**
    - 购买转化率 p=0.6098
    - 加购率 p=0.9766
3.  **最终业务结论**
    加购/购买意向排序的新策略，对比原有浏览排序策略**无显著业务提升**。
    业务侧建议维持原有推荐排序策略，无需全量上线新策略；后续可优化策略权重、缩小用户分析范围后重启实验验证。
