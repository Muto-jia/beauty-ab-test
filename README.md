# 美妆电商用户行为 A/B 测试
基于美妆用户行为数据，对比「浏览排序」和「加购/购买排序」对转化效果的影响。

## 项目背景
- 数据来源：阿里云美妆用户行为数据集
- 实验策略：推荐商品排序规则对比
- 实验分组：A组（浏览排序）、B组（加购+购买意向排序）

## 核心指标
- 购买转化率
- 加购率
- 人均GMV
- 人均浏览商品数

## 实验结论
- 两组策略**所有指标差异均不显著（p>0.05）**
- 新策略未带来显著提升，建议维持原有策略

## 技术栈
- MySQL：数据存储与清洗
- Python：指标计算、AB分流、统计检验、可视化
- 统计方法：卡方检验、t检验
